O programa em questão foi codado e compilado através da plataforma GDB Online.

No programa conseguimos executar os itens 1, 2, 3 e 4. 
Cada thread ficou responsável pela a execução de uma função no código e para 
resolver o problema de concorrência utilizamos mutex.

Como prova disso, assim que executado, o programa indica que as duas transferências
propostas foram executadas com sucesso, o que é confirmado pelo resultado indicado
no saldo das contas.
